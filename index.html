One-time setup — takes 3 minutes. After this, every trainer sees live changes instantly.
1
Go to console.firebase.google.com → Create project (any name, e.g. skillinabox) → Disable Google Analytics (not needed)
2
In your project → Build → Realtime Database → Create Database → Start in test mode → Choose any location → Click Enable
3
Go to Project Settings (⚙ gear icon) → General → scroll to "Your apps" → Click </> Web → Register app → Copy the databaseURL value (looks like https://your-project-default-rtdb.firebaseio.com) and your apiKey
