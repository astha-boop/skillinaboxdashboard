1
Go to console.firebase.google.com → Create project (any name, e.g. skillinabox) → Disable Google Analytics (not needed)
2
In your project → Build → Realtime Database → Create Database → Start in test mode → Choose any location → Click Enable
3
Go to Project Settings (⚙ gear icon) → General → scroll to "Your apps" → Click </> Web → Register app → Copy the databaseURL value (looks like https://your-project-default-rtdb.firebaseio.com) and your apiKey

🔧 Fix Firebase Rules
⚠️ Your data is not being saved!
Firebase "test mode" database rules expire after 30 days. This is the most common reason data vanishes on reload.
Follow these steps to fix it in ~2 minutes:
1 Open console.firebase.google.com → select your project (skillinabox-63ad8)
2 In the left sidebar: Build → Realtime Database → Rules tab
3 Replace all the text in the rules editor with this:
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}
4 Click the Publish button → come back here and refresh the page. Your data will load.
